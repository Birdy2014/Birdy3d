Sets the position and size of all widgets in the list.

The size of the widgets is set so that they match the size given in the size parameter. The size parameter can never exceed the value returned by minimal_size(). An overloaded function should call Widget::arrange(glm::ivec2, glm::ivec2) on each child widget.

Implemented in Birdy3d::ui::DirectionalLayout, Birdy3d::ui::StaticGridLayout, Birdy3d::ui::DynamicGridLayout, Birdy3d::ui::MaxLayout, and Birdy3d::ui::AbsoluteLayout.
